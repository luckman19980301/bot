# build-a-bot

## Project setup
```
git clone https://github.com/kwood15/build-a-bot.git
```

### Install dependencies
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
